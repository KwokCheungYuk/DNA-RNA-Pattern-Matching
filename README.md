# README

This is a project for pattern matching in degenerate DNA/RNA.  The core matching algorithm is proposed in[1].

The file **Operating instructions.md** explains how to use this project.



**Reference**

[1] C.S. Iliopoulos, L. Mouchard and M.S. Rahman, "A new approach to pattern matching in degenerate DNA/RNA sequences and distributed pattern matching," Mathematics in Computer Science, vol. 1, pp. 557-569, 2008.
